## Сценарии ручной проверки

### С1. Переход в уведомления (US1, FR-001, FR-002)

1. Открыть вкладку «Главная».
2. Убедиться: на колокольчике badge = 3 (мок).
3. Нажать колокольчик → экран «Уведомления».
4. Нажать «Назад» → возврат на «Главную».

---

### С2. Список и индикация (US1, FR-003–FR-004, SC-003)

1. Открыть «Уведомления».
2. Заголовок «Уведомления» отображается полностью (без кнопки mark-all в AppBar).
3. Видны элементы списка; непрочитанные визуально отличаются.
4. При непрочитанных — внизу экрана плавающая кнопка «Отметить все прочитанным».

---

### С3. Отметить все прочитанным (US1, FR-008, FR-009, SC-002)

1. На экране «Уведомления» нажать «Отметить все прочитанным» (FAB внизу).
2. Все элементы отображаются как прочитанные; FAB скрывается.
3. Вернуться на «Главную» — badge скрыт или 0.
4. Снова открыть список — FAB mark-all отсутствует.

---

### С4. Деталь уведомления (US1, FR-005–FR-007)

1. Сбросить мок (перезапуск приложения) для непрочитанных.
2. Открыть непрочитанное уведомление из списка.
3. Проверить: заголовок, время, полное содержимое, кнопка «Назад».
4. «Назад» → список; элемент помечен прочитанным.
5. «Главная» → badge уменьшился на 1.

---

### С5. Пустой список (Edge case)

1. Отметить все прочитанными (или мок с пустым списком при настройке).
2. При отсутствии уведомлений — текст «Уведомлений пока нет».

---

### С6. Главная: баннеры и отступ (US2, FR-010–FR-011, SC-004)

1. Вкладка «Главная».
2. Между шапкой и баннерами — заметный отступ.
3. Три баннера; активный на всю ширину, по краям видны части соседних.
4. Подождать ~5 с — автопереход к следующему баннеру.
5. Свайп влево/вправо — циклическая прокрутка по кругу.

---

### С7. Блок «Связаться» (US2, FR-012)

1. На «Главной» найти серую кнопку «Связаться» с иконкой телефона (под баннерами).
2. Нажать → на устройстве инициируется звонок `+78125645548`.

---

### С8. Вкладка «Акции» (US3, FR-013)

1. Tab Bar: третья вкладка — «Акции» (не «Акции и новости»).
2. Открыть вкладку — заголовок экрана «Акции».

## Проверка OpenAPI и моков

Сверить с [contracts/openapi.yaml](./contracts/openapi.yaml):

| Эндпоинт | Ожидание мока |
|----------|----------------|
| `GET /notifications` | ≥ 3 items, 3 с `isRead: false` |
| `GET /notifications/{id}` | деталь по id |
| `POST /notifications/{id}/read` | `isRead` → true |
| `POST /notifications/read-all` | все `isRead: true` |
| `GET /home/banners` | ровно 3 баннера |
