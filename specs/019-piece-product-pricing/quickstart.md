# Quickstart: Штучный товар, цена за кг и старая цена

**Фича**: [spec.md](./spec.md) | **План**: [plan.md](./plan.md) | **Модель**: [data-model.md](./data-model.md)

Руководство по проверке фичи после реализации.

## Предварительные требования

- Flutter SDK (stable), Dart 3.x
- Репозиторий с выполненными задачами из `tasks.md` (после `/speckit-tasks`)
- Мок-API включён (`useMockApi = true` в `lib/core/network/providers.dart`)
- `openapi/openapi.yaml` содержит `pricePerKgRub`, `pieceProduct`, `oldPriceRub` в ProductSummary/Detail
- Моки возвращают товары с разными комбинациями полей (см. [research.md](./research.md) §7)

## Установка и запуск

```bash
flutter pub get
flutter run
```

## Сценарии ручной проверки

### С1. Старая цена на кнопке в каталоге (US1, FR-002)

1. Открыть «Каталог» → категория «Рыба».
2. Найти товар id **1000** (мок: `oldPriceRub` на 150 ₽ выше `priceRub`).
3. Убедиться, что товар **не** в корзине.
4. На жёлтой кнопке слева — **зачёркнутая** старая цена, справа — текущая и «+».
5. Нажать «+» — панель «− 1 × [текущая] +» **без** старой цены.

### С2. Нет старой цены при равенстве (FR-003)

1. Найти товар без скидки (например, id **1003**, `oldPriceRub == priceRub`).
2. Кнопка показывает только «[текущая] +», без зачёркивания.

### С3. Цена за кг на карточке (US2, FR-005–FR-006)

1. Товар **1000** или **1002** — под весом видна строка вида «2 400 ₽/кг» (или другая
   ненулевая из мока).
2. Товар с `pricePerKgRub: 0` — строки цены за кг **нет**.

### С4. Страница товара (US1 + US2)

1. Открыть деталь товара **1000**.
2. Под весом — цена за кг (если > 0).
3. Внизу, пока товар не в корзине — кнопка со зачёркнутой старой ценой.
4. Добавить в корзину — нижняя панель без старой цены.

### С5. Товары недели (FR-002, FR-005)

1. «Главная» → блок «Товары недели».
2. Карточки с теми же правилами старой цены и цены/кг, что в каталоге.

### С6. Спеццена + старая цена (Edge case)

1. Товар **1001** (`special: true`, старая = обычная цена выше спеццены в моке).
2. Бейдж «СПЕЦЦЕНА» на месте; на кнопке — зачёркнутая старая и спеццена, если
   `oldPriceRub > priceRub`.

### С7. Штучный товар (US3)

1. Товар **1002** (`pieceProduct: true`).
2. Карточка и деталь загружаются без ошибок; добавление в корзину работает.

### С8. Боевой API (опционально)

1. `useMockApi = false` в `providers.dart`.
2. `flutter run` — каталог с `https://trout.osetrovich.ru/v1`.
3. Цены и поля соответствуют ответу сервера.

## Критерии готовности

- [ ] Парсинг JSON не падает на обязательных `pricePerKgRub` и `pieceProduct`
- [ ] 100% товаров с `oldPriceRub > priceRub` и qty=0 показывают зачёркнутую цену на кнопке
- [ ] 0% товаров с `oldPriceRub ≤ priceRub` показывают зачёркивание
- [ ] Цена/кг только при `pricePerKgRub > 0` на карточке и детали
- [ ] Бейджи «Акция», «СПЕЦЦЕНА», «Товар недели» без регрессий
- [ ] `flutter test` и `flutter analyze` без ошибок
