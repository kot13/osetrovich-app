# Quickstart: Колокольчик уведомлений на всех вкладках Tab Bar

**Фича**: [spec.md](./spec.md) | **План**: [plan.md](./plan.md) | **Модель**: [data-model.md](./data-model.md)

Руководство по проверке фичи после реализации.

## Предварительные требования

- Flutter SDK (stable), Dart 3.x
- Репозиторий с выполненными задачами из `tasks.md`
- Мок-API включён (`useMockApi = true` в `lib/core/network/providers.dart`)
- Реализованы фичи: уведомления (`002`, `011`), shell (`001`)

## Установка и запуск

```bash
flutter pub get
flutter run
```

## Сценарии ручной проверки

### С1. Колокольчик на всех вкладках (US1, FR-001–FR-003)

1. Войти под любым демо-номером (например, `+79001111111`).
2. Последовательно открыть: «Главная» → «Каталог» → «Акции» → «Корзина» → «Профиль».
3. На каждом корневом экране в правой части шапки — иконка колокольчика.
4. При наличии непрочитанных (мок по умолчанию) — красный бейдж с числом.

---

### С2. Открытие с «Корзины» и возврат (US2, FR-004–FR-006)

1. Перейти на вкладку «Корзина» (нижний Tab Bar подсвечен «Корзина»).
2. Нажать колокольчик → открывается экран «Уведомления».
3. Убедиться: нижний Tab Bar **по-прежнему** на «Корзине» (не переключился на «Главную»).
4. Нажать «Назад» → снова корневой экран «Корзина».

---

### С3. Открытие с «Каталога» и деталь уведомления (US2)

1. Перейти на «Каталог», нажать колокольчик.
2. Открыть любое уведомление из списка.
3. «Назад» → список уведомлений.
4. «Назад» → корневой «Каталог»; активная вкладка — «Каталог».

---

### С4. Повторить для «Акции» и «Профиль» (US2, SC-004)

Повторить С2/С3 для вкладок «Акции» и «Профиль» (включая гостевой профиль без авторизации —
колокольчик виден).

---

### С5. Синхронизация бейджа (US3, FR-007, SC-003)

1. Убедиться, что бейдж показывает N > 0 (например, на «Главной»).
2. Перейти на «Каталог» — то же число N.
3. Открыть уведомления, нажать «Отметить все прочитанным».
4. Вернуться назад, переключиться на «Профиль» — бейдж отсутствует.

---

### С6. Вложенные экраны без колокольчика (Edge Cases)

1. Из «Каталога» открыть карточку товара — в шапке **нет** колокольчика (только стандартная
   навигация детали).
2. Из «Акции» открыть статью — колокольчика в шапке нет.

---

### С7. Регрессия диплинка (Assumptions)

1. Симулировать переход на `osetrovich://notifications` (или через push tap из фичи 017).
2. Открывается список уведомлений на ветке `/home` — поведение без регрессий.

---

## Справочно: мок непрочитанных

По умолчанию `MockApiClient` возвращает ненулевой `unread-count` для авторизованного
пользователя. После «Отметить все прочитанным» счётчик обнуляется до перезапуска приложения
(мутабельное состояние мока).
