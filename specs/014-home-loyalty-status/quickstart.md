# Quickstart: Блок статуса лояльности на главной

**Фича**: [spec.md](./spec.md) | **План**: [plan.md](./plan.md) | **Модель**: [data-model.md](./data-model.md)

Руководство по проверке фичи после реализации.

## Предварительные требования

- Flutter SDK (stable), Dart 3.x
- Репозиторий с выполненными задачами из `tasks.md`
- Мок-API включён (`useMockApi = true` в `lib/core/network/providers.dart`)
- Реализованы фичи: авторизация (`001`), профиль (`003`), главная (`007`)

## Установка и запуск

```bash
flutter pub get
flutter run
```

## Сценарии ручной проверки

### С1. Гость — кнопка «Авторизоваться» (US1, FR-001–FR-003)

1. Открыть приложение без авторизации.
2. Перейти на вкладку «Главная».
3. Убедиться: под баннерами **нет** кнопки «Связаться».
4. Убедиться: отображается кнопка **«Авторизоваться»**.
5. Убедиться: внизу экрана **нет** баннера «Авторизуйтесь».
6. Нажать «Авторизоваться» — открывается экран ввода телефона.

---

### С2. Авторизованный со статусом, скидкой и картой (US2, FR-005–FR-008)

1. Войти под `+79001111111` (мок: Premium, скидка 10 %, карта `1234567890`).
2. Открыть «Главную».
3. Убедиться: под баннерами блок лояльности.
4. В блоке: название **Premium**, строка **Скидка 10%**, номер карты.
5. Кнопки «Авторизоваться» и «Связаться» в этой зоне нет.

---

### С3. Статус без скидки (US2, FR-007)

1. Войти под `+79002222222` (мок: VIP, скидка 0 %).
2. На «Главной» — блок со статусом **VIP**.
3. Строка со скидкой **отсутствует**.

---

### С4. Авторизован без статуса (US2, FR-009)

1. Войти под `+79003333333` (мок: `loyaltyStatus = null`).
2. На «Главной» зона под баннерами **пустая** (нет auth, нет loyalty).

---

### С5. Pull-to-refresh (US2, FR-010)

1. Войти под `+79001111111`, открыть «Главную», увидеть блок лояльности.
2. Потянуть список вниз для обновления.
3. Блок остаётся с актуальными данными; ошибок загрузки нет.

---

### С6. Выход из аккаунта (US3, FR-011)

1. Будучи авторизованным со статусом, открыть «Главную» — виден блок лояльности.
2. Выйти через «Профиль».
3. Вернуться на «Главную» — блок исчез, появилась «Авторизоваться».

---

### С7. «Связаться» в других местах (Assumptions)

1. Без авторизации: в профиле ContactBlock «Связаться» по-прежнему доступен.
2. С активным заказом: в блоке заказа на главной — «Связаться с оператором».
